# Quick Start

This guide gets you from a blank launch to a saved beat.

## 1. Start the App

Open GrimoHex on iPhone or iPad. A splash screen appears briefly, then the main performance screen loads. The app restores the last saved project if one exists. Otherwise it starts a blank project and assigns the bundled factory kit to audio tracks T1-T8.

The default kit is:

| Track | Default sound |
|---|---|
| T1 | KICK |
| T2 | SNARE |
| T3 | HAT |
| T4 | CLAP |
| T5 | COWBELL |
| T6 | SNARE |
| T7 | HAT |
| T8 | CLAP |

## 2. Play the Pads

When recording is not armed, the lower 4x4 pad area is in track-pad mode.

- Tap T1-T8 to preview audio tracks.
- Tap M1-M8 to select or trigger MIDI tracks.
- The selected audio track is highlighted in orange.
- MIDI tracks use cyan highlighting.

## 3. Add Steps

Tap **GRID** or **REC** to arm grid recording. The lower pad area changes to a 16-step page for the selected track.

- Tap a step to turn it on or off.
- Use **PREV** and **NEXT** to move through pages when the pattern is longer than 16 steps.
- Active steps show track-colored fill.
- Steps with locks, conditions, retrig, or microtiming show markers.

Press **Play** to hear the pattern.

## 4. Adjust Tempo, Swing, and Length

Tap the top-bar status fields:

- **TEMPO** opens a BPM editor from 30.0 to 300.0 BPM.
- **SWING** opens a swing editor from 50 to 80.
- **LEN** opens pattern length from 1 to 64 steps.
- **PATTERN** opens bank and pattern selection plus pattern copy, paste, clear, and duplicate actions.

## 5. Shape the Sound

Use the page selector to open parameter pages:

- **STEP**: note, velocity, length, probability/condition, microtiming, retrig, tune.
- **SOURCE**: machine, tune, playback mode or slice controls, sample slot, start, end, loop, balance.
- **FILTER**: filter envelope and filter controls.
- **LEVEL**: amp envelope, overdrive, sends, pan, volume.
- **MOTION**: LFO speed, multiplier, fade, destination, wave, start, mode, depth.

Drag an encoder vertically to change a value. Double-tap an encoder to reset it.

## 6. Add a Parameter Lock

In the step grid, long-press a step. A step actions panel appears.

While a step is held or selected in grid edit, parameter edits can become step-specific locks. Locked parameters show a yellow **LOCK** tag in the parameter cell.

Use the step actions panel to copy, paste, clear, add a fill condition, enable retrig, or set a playback mode lock for that step.

## 7. Load or Record a Sample

Open the **SAMPLE** page.

- Tap **LOAD** to open the sound browser and assign a bundled or imported sample to the selected track.
- Tap **RECORD SAMPLE** to open Sample Record.
- Tap **SLICE** to toggle slice machine mode.
- Tap **AUTO** to create transient slice boundaries for the current sample.

## 8. Save the Project

Open the app menu or **SETTINGS** page and use:

- **SAVE** to save the current project.
- **SAVE AS** to copy the project under a new name.
- **NEW** to create a new blank project after saving the current one.
- **LOAD** to open a saved project.

GrimoHex also autosaves project changes with a short debounce.
