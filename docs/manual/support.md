# Support

## What to Include in a Support Request

For the fastest help, include:

- Device model.
- iOS or iPadOS version.
- GrimoHex app version and build number.
- Whether you are using built-in speaker, headphones, Bluetooth, or USB audio.
- Whether the problem happens in a new project.
- Steps to reproduce the issue.
- For sample import issues, the file type and approximate file length.
- For MIDI issues, the receiving app or device and expected channel.

## Common Support Categories

- No audio output.
- Sample import or assignment.
- Sample recording permission or input level.
- Project save/load.
- MIDI routing.
- Performance or route changes.
- Unexpected UI state after switching modes.

## App Version

The current Info.plist sets:

- Short version: 1.0.
- Build version: 1.

## Local Data

Projects and samples are stored locally by the app. Deleting the app may remove local projects and imported or recorded samples unless they have been exported or otherwise backed up outside the app.
